## 🚀 الاستخدام

### استخدام تلقائي
النظام يعمل تلقائياً عند كل محادثة:
- تحليل جودة الرد
- تحسين الرد إذا لزم الأمر
- حفظ العينات عالية الجودة فقط

### استخدام يدوي

#### 1. تحليل جودة البيانات التدريبية
```python
from training_enhancer import get_training_enhancer

enhancer = get_training_enhancer()
analysis = enhancer.analyze_training_data(user_id="user-123")
print(analysis)
```

#### 2. الحصول على عينات عالية الجودة
```python
samples = enhancer.get_high_quality_samples(user_id="user-123", limit=100)
```

#### 3. إنشاء ملف تدريبي
```python
count = enhancer.generate_training_dataset(
    output_file='training_data.jsonl',
    user_id="user-123",
    limit=500
)
```

#### 4. تصدير للتدريب
```python
from training_data_exporter import get_training_exporter

exporter = get_training_exporter()

# تصدير لـ OpenAI
exporter.export_for_openai_finetuning(
    output_file='openai_training.jsonl',
    limit=1000
)

# تصدير لـ ChatGPT
exporter.export_for_chatgpt_training(
    output_file='chatgpt_training.jsonl',
    limit=1000
)
```

## 📊 معايير الجودة

### درجة الجودة (Quality Score)
- **0.8 - 1.0**: ممتاز (Excellent)
- **0.6 - 0.8**: جيد (Good)
- **0.4 - 0.6**: متوسط (Fair)
- **0.0 - 0.4**: ضعيف (Poor)

### العوامل المؤثرة في الجودة:
1. **طول الرسالة والرد** (25%)
2. **وجود نتائج بحث** (20%)
3. **وجود intent و sentiment** (15%)
4. **confidence score** (10%)
5. **جودة المحتوى** (20%)
6. **وجود tags** (10%)
