# 🎯 Backlog للمهندس - قائمة المهام والميزات الجديدة

## 📋 نظرة عامة

هذا الملف يحتوي على قائمة المهام والميزات المطلوبة لتطوير Eva Chatbot. كل ميزة تحتوي على وصف تفصيلي ومتطلبات واضحة.

## 🚀 الميزات الجديدة المطلوبة (20+ ميزة)

### 1. 🔐 نظام المصادقة والمستخدمين
**الأولوية**: عالية  
**التعقيد**: متوسط  
**الوصف**: 
- تسجيل دخول/تسجيل مستخدمين
- ملفات شخصية للمستخدمين
- حفظ تاريخ المحادثات لكل مستخدم
- إعدادات شخصية (اللغة، الوضع المفضل)

**المتطلبات**:
- جدول `users` في قاعدة البيانات
- JWT tokens للمصادقة
- API endpoints: `/api/auth/login`, `/api/auth/register`, `/api/user/profile`
- Frontend: صفحات تسجيل الدخول والملف الشخصي
