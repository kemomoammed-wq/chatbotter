# 🚀 كيفية التشغيل السريع

## خطوة واحدة فقط! 

### Windows:
```bash
START.bat
```

هذا كل شيء! الملف سيقوم بـ:
1. ✅ التحقق من جميع المتطلبات (Node.js, Python, npm)
2. ✅ إعداد البيئة الافتراضية Python تلقائياً
3. ✅ تثبيت جميع المتطلبات
4. ✅ بناء React Frontend تلقائياً
5. ✅ تشغيل FastAPI Backend
6. ✅ فتح المتصفح تلقائياً

## 🌐 الروابط بعد التشغيل:

- **Frontend**: http://localhost:8000
- **API**: http://localhost:8000/api
- **Health Check**: http://localhost:8000/api/health

## 📝 ملاحظات:

- السيرفر يعمل في نافذة منفصلة
- لإيقاف السيرفر: اضغط `Ctrl+C` في نافذة السيرفر
- أو أغلق نافذة "Eva Chatbot - Backend"

## 🔧 المتطلبات:

- Windows 10/11
- Node.js (الإصدار 16 أو أحدث)
- Python 3.8 أو أحدث
- اتصال بالإنترنت (لأول مرة فقط)

## ❓ مشاكل شائعة:

### البورت 8000 مستخدم:
```bash
# أوقف السيرفرات القديمة
taskkill /F /IM uvicorn.exe
```

### React build فشل:
```bash
cd eva-wise-chat-buddy-main
npm install
npm run build
cd ..
```

### Python packages مفقودة:
```bash
call myenv\Scripts\activate.bat
pip install -r requirements_advanced.txt
```
