# ⚡ بدء سريع - تشغيل التطبيق

## 🚀 الطريقة الأسهل (موصى بها)

### شغّل كل شيء بضغطة واحدة:
```bash
START_ALL.bat
```

**هذا كل شيء!** 🎉

---

## 📍 بعد التشغيل

افتح المتصفح على:
- **http://localhost:8000** - التطبيق الكامل
- **http://localhost:8000/api/health** - التحقق من الحالة

---

## 🛑 للإيقاف

اضغط `Ctrl+C` في نافذة السيرفر
أو شغّل:
```bash
STOP.bat
```

---

## 📚 للمزيد من التفاصيل

راجع ملف `HOW_TO_RUN.md` للتفاصيل الكاملة.

---

**جاهز للاستخدام!** ✨
