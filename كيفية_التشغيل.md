# 🚀 كيفية تشغيل التطبيق - خطوة بخطوة

## 📋 المتطلبات الأساسية

1. ✅ Python 3.8+ مثبت
2. ✅ Node.js 18+ مثبت (للبناء الأول فقط)
3. ✅ البيئة الافتراضية `myenv` موجودة

---

## 🎯 الطريقة السريعة (للتشغيل اليومي)

### خطوة 1: فتح PowerShell في مجلد المشروع

```powershell
cd C:\Users\WIN11-2025-\Desktop\chatot
```

### خطوة 2: تفعيل البيئة الافتراضية

```powershell
.\myenv\Scripts\activate
```

سترى `(myenv)` في بداية السطر - هذا يعني أن البيئة مفعلة ✅

### خطوة 3: تشغيل السيرفر

```powershell
uvicorn fastapi_app:app --host 0.0.0.0 --port 8000
```

### خطوة 4: فتح المتصفح

افتح المتصفح واذهب إلى: **http://localhost:8000**

---

## 🔧 الطريقة المتقدمة (تشغيل في الخلفية)

إذا أردت تشغيل السيرفر في الخلفية بدون إبقاء نافذة PowerShell مفتوحة:

```powershell
cd C:\Users\WIN11-2025-\Desktop\chatot
.\myenv\Scripts\activate
Start-Process -NoNewWindow -PassThru -FilePath .\myenv\Scripts\uvicorn.exe -ArgumentList 'fastapi_app:app','--host','0.0.0.0','--port','8000'
```

---

## 🛑 إيقاف السيرفر

### إذا كان السيرفر يعمل في نافذة PowerShell:
اضغط `Ctrl + C` في نافذة PowerShell

### إذا كان السيرفر يعمل في الخلفية:
```powershell
Get-Process uvicorn | Stop-Process -Force
```

---

## 🔍 التحقق من أن السيرفر يعمل

افتح PowerShell جديد واكتب:

```powershell
Invoke-RestMethod -Uri http://localhost:8000/api/health
```

إذا رأيت:
```
status chatbot_loaded
------ --------------
ok               True
```

يعني السيرفر يعمل بنجاح! ✅

## 📝 ملاحظات مهمة

### 1. البورت 8000 مشغول؟
إذا ظهرت رسالة خطأ تقول أن البورت 8000 مشغول:

```powershell
# أوقف أي عملية تستخدم البورت 8000
Get-Process uvicorn | Stop-Process -Force

# أو استخدم بورت آخر
uvicorn fastapi_app:app --host 0.0.0.0 --port 8001
```

### 2. الواجهة React لم تظهر؟
تأكد من أن مجلد `eva-wise-chat-buddy-main/dist` موجود. إذا لم يكن موجوداً:

```powershell
cd eva-wise-chat-buddy-main
npm install
npm run build
cd ..
```

### 3. خطأ في استيراد المكتبات؟
تأكد من تفعيل البيئة الافتراضية أولاً:

```powershell
.\myenv\Scripts\activate
```

ثم ثبت المكتبات المفقودة:

```powershell
pip install -r requirements_advanced.txt
```

## 📱 الـ API Endpoints المتاحة

بعد تشغيل السيرفر، يمكنك استخدام:

- **الواجهة:** http://localhost:8000
- **صحة السيرفر:** http://localhost:8000/api/health

- **المحادثات:** GET http://localhost:8000/api/conversations?user_id=xxx

## 🆘 حل المشاكل الشائعة

### المشكلة: "uvicorn: command not found"
**الحل:** تأكد من تفعيل البيئة الافتراضية:
```powershell
.\myenv\Scripts\activate
```

### المشكلة: "ModuleNotFoundError"
**الحل:** ثبت المكتبات المفقودة:
```powershell
pip install fastapi uvicorn[standard] python-multipart
```

### المشكلة: "Port 8000 already in use"
**الحل:** أوقف العملية القديمة:
```powershell
Get-Process uvicorn | Stop-Process -Force
```

### المشكلة: الواجهة لا تظهر
**الحل:** قم ببناء React:
```powershell
cd eva-wise-chat-buddy-main
npm run build
cd ..
```
