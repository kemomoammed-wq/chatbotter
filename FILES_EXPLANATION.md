# 📚 شرح شامل لجميع الملفات

## 🎯 الملفات الأساسية (11 ملف)

### 1. `RUN.bat` ⭐ **ملف التشغيل الوحيد**

**ما يفعله:**

- يشغل FastAPI على المنفذ 8000
- يخدم React build

**الاستخدام:** اضغط نقرتين عليه

### 2. `STOP.bat` ⭐ **ملف الإيقاف**
**الوظيفة:** إيقاف جميع السيرفرات
**الاستخدام:** اضغط نقرتين لإيقاف كل شيء

---

### 3. `fastapi_app.py` ⭐ **السيرفر الرئيسي**
**الوظيفة:** FastAPI server مع جميع API endpoints

**API Endpoints:**
- `GET /api/health` - فحص الحالة
- `POST /api/chat` - محادثة مع الشات بوت
- `POST /api/search` - البحث في الإنترنت
- `GET /api/training/quality` - جودة البيانات التدريبية
- `GET /api/training/high-quality` - عينات عالية الجودة
- `POST /api/training/export` - تصدير البيانات
- `GET /api/training/statistics` - إحصائيات

**الاعتماديات:**
- `core_integration` - التكامل المركزي
- `web_search_service` - البحث
- `database` - قاعدة البيانات
- `response_quality_improver` - تحسين الردود

---

### 4. `core_integration.py` ⭐ **نظام التكامل المركزي**
**الوظيفة:** ربط جميع المكونات ببعضها

**المكونات:**
1. Database - قاعدة البيانات
2. Chatbot - الشات بوت
3. Web Scraper - استخراج البيانات
4. Web Search Service - البحث في الإنترنت
5. Knowledge Base - قاعدة المعرفة
6. Conversation Memory - ذاكرة المحادثة
7. Vector Database - قاعدة بيانات ناقلات (اختياري)
8. Analytics - التحليلات (اختياري)

**الاستخدام:** يتم استدعاؤه تلقائياً من `fastapi_app.py`

---

### 5. `advanced_chatbot_integration.py` ⭐ **الشات بوت الرئيسي**
**الوظيفة:** معالجة الرسائل وتوليد الردود

**الميزات:**
- معالجة اللغة العربية والإنجليزية
- RAG (Retrieval Augmented Generation)
- البحث في قاعدة البيانات الطبية
- البحث في الإنترنت
- تحسين الردود تلقائياً
- حفظ ذاكرة المحادثة

**الدوال الرئيسية:**
- `process_message()` - معالجة الرسالة
- `_generate_llm_response()` - توليد الرد

**الاعتماديات:**
- `nlp_preprocessing` - معالجة النصوص
- `llm_integration` - نماذج LLM
- `conversation_memory` - ذاكرة المحادثة
- `rag_enhancer` - تحسين البحث

---

### 6. `database.py` ⭐ **قاعدة البيانات**
**الوظيفة:** إدارة قاعدة البيانات SQLite

**الجداول:**
- `conversations` - المحادثات المحفوظة
- `training_samples` - عينات التدريب
- `medical_data` - البيانات الطبية والصيدلانية
- `scraped_data` - البيانات المستخرجة من المواقع
- `users` - المستخدمين

**الدوال الرئيسية:**
- `save_conversation()` - حفظ محادثة
- `save_training_sample()` - حفظ عينة تدريبية
- `search_medical_data()` - البحث في البيانات الطبية
- `search_scraped_data()` - البحث في البيانات المستخرجة

**الاعتماديات:** `sqlalchemy`

---

### 7. `web_search_service.py` ⭐ **خدمة البحث في الإنترنت**
**الوظيفة:** البحث في الإنترنت واستخراج البيانات

**الميزات:**
- البحث في DuckDuckGo
- البحث في مواقع الصيدليات العربية
- استخراج البيانات من النتائج
- دعم اللغة العربية

**الدوال الرئيسية:**
- `search_web()` - البحث العام
- `search_pharmacy_sites()` - البحث في مواقع الصيدليات
- `search_and_scrape()` - البحث واستخراج البيانات

**الاعتماديات:**
- `ddgs` - DuckDuckGo Search
- `web_scraper` - استخراج البيانات

---

### 8. `web_scraper.py` ⭐ **استخراج البيانات من المواقع**
**الوظيفة:** استخراج البيانات من صفحات الويب

**الميزات:**
- دعم مواقع متعددة (ويب طب، ويكيبيديا، صيدليات)
- استخراج العنوان، الوصف، المحتوى
- استخراج السعر (للمنتجات)
- معالجة خاصة لكل موقع
- معالجة الترميز العربي

**الدوال الرئيسية:**
- `scrape_url()` - استخراج بيانات من رابط
- `_extract_title()` - استخراج العنوان
- `_extract_content()` - استخراج المحتوى
- `_extract_price()` - استخراج السعر

**الاعتماديات:**
- `beautifulsoup4` - تحليل HTML
- `requests` - طلبات HTTP

---

### 9. `pharmacy_sites_config.py` ⭐ **إعدادات مواقع الصيدليات**
**الوظيفة:** قائمة مواقع الصيدليات العربية

**المواقع المدعومة:**
- صحتك (sehetak.com) - مصر
- دواي (dawaey.com) - مصر
- صيدلية النهدي (nahdionline.com) - السعودية
- ويب طب (webteb.com) - عربي
- الطبي (altibbi.com) - عربي
- وغيرها

**الدوال:**
- `get_pharmacy_site_config()` - الحصول على إعدادات موقع
- `is_pharmacy_site()` - التحقق من موقع صيدلية
- `get_search_urls()` - الحصول على روابط البحث

**الاستخدام:** يتم استدعاؤه تلقائياً من `web_search_service.py`

### 10. `nlp_preprocessing.py` ⭐ **معالجة النصوص**
**الوظيفة:** معالجة وتحليل النصوص

**الميزات:**
- تنظيف النصوص
- Tokenization (تقسيم النص)
- إزالة Stopwords (الكلمات الشائعة)
- Lemmatization (تصريف الكلمات)
- تحليل المشاعر
- اكتشاف اللغة

**الدوال الرئيسية:**
- `preprocess_pipeline()` - معالجة كاملة
- `tokenize()` - تقسيم النص
- `analyze_sentiment()` - تحليل المشاعر
- `detect_language()` - اكتشاف اللغة

**الاعتماديات:**
- `nltk` (اختياري) - معالجة النصوص المتقدمة
- `spacy` (اختياري) - NLP متقدم
- `vaderSentiment` (اختياري) - تحليل المشاعر

**ملاحظة:** يعمل بدون nltk (fallback mode)

## 📦 الملفات المساعدة

### `training_data_exporter.py`
**الوظيفة:** تصدير البيانات التدريبية
**الصيغ:** OpenAI Fine-tuning, ChatGPT Training

### `conversation_memory.py`
**الوظيفة:** حفظ ذاكرة المحادثات
**الميزات:** حفظ تاريخ المحادثة للسياق

### `llm_integration.py`
**الوظيفة:** التكامل مع نماذج LLM
**الميزات:** دعم OpenAI, HuggingFace, Ollama

### `ai_knowledge_base.py`
**الوظيفة:** قاعدة معرفة للشات بوت
**المحتوى:** معلومات طبية وصيدلانية

### `scraper_manager.py`
**الوظيفة:** إدارة استخراج البيانات
**الميزات:** Batch processing, أولويات

### `vector_database.py`
**الوظيفة:** قاعدة بيانات ناقلات للبحث الدلالي
**الحالة:** اختياري (يعمل بدونها)

### `auth.py`
**الوظيفة:** نظام المصادقة
**الميزات:** تسجيل الدخول، التسجيل
