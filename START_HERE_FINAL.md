## 📁 هيكل المشروع

### 🔧 ملفات التشغيل (2 ملف فقط)

#### `RUN.bat` ⭐ **الملف الوحيد للتشغيل**
- **الوظيفة:** تشغيل التطبيق الكامل
- **ما يفعله:**
  - يتحقق من Python
  - ينشئ البيئة الافتراضية
  - يثبت المتطلبات تلقائياً
  - يشغل FastAPI على المنفذ 8000
- **الاستخدام:** اضغط نقرتين عليه

#### `STOP.bat` ⭐ **ملف الإيقاف**
- **الوظيفة:** إيقاف جميع السيرفرات
- **الاستخدام:** اضغط نقرتين لإيقاف كل شيء

---

### 🐍 ملفات Python الأساسية (13 ملف)

#### 1. `fastapi_app.py` ⭐⭐⭐ **السيرفر الرئيسي**
**الوظيفة:** FastAPI server مع جميع API endpoints

**API Endpoints:**
- `GET /api/health` - فحص الحالة
- `POST /api/chat` - محادثة مع الشات بوت
- `POST /api/search` - البحث في الإنترنت
- `GET /api/training/quality` - جودة البيانات التدريبية
- `POST /api/training/export` - تصدير البيانات

**الاعتماديات:** `core_integration`, `web_search_service`, `database`

---

#### 2. `core_integration.py` ⭐⭐⭐ **نظام التكامل المركزي**
**الوظيفة:** ربط جميع المكونات ببعضها

**المكونات:**
- Database - قاعدة البيانات
- Chatbot - الشات بوت
- Web Scraper - استخراج البيانات
- Web Search Service - البحث
- Knowledge Base - قاعدة المعرفة
- Conversation Memory - ذاكرة المحادثة

**الاستخدام:** يتم استدعاؤه تلقائياً

---

#### 3. `advanced_chatbot_integration.py` ⭐⭐⭐ **الشات بوت الرئيسي**
**الوظيفة:** معالجة الرسائل وتوليد الردود

**الميزات:**
- معالجة اللغة العربية والإنجليزية
- RAG (Retrieval Augmented Generation)
- البحث في قاعدة البيانات والإنترنت
- تحسين الردود تلقائياً

**الدوال الرئيسية:**
- `process_message()` - معالجة الرسالة
- `_generate_llm_response()` - توليد الرد

---

#### 4. `database.py` ⭐⭐⭐ **قاعدة البيانات**
**الوظيفة:** إدارة قاعدة البيانات SQLite

**الجداول:**
- `conversations` - المحادثات
- `training_samples` - عينات التدريب
- `medical_data` - البيانات الطبية
- `scraped_data` - البيانات المستخرجة
- `users` - المستخدمين

**الدوال الرئيسية:**
- `save_conversation()` - حفظ محادثة
- `save_training_sample()` - حفظ عينة تدريبية
- `search_medical_data()` - البحث في البيانات الطبية

---

#### 5. `web_search_service.py` ⭐⭐⭐ **خدمة البحث في الإنترنت**
**الوظيفة:** البحث في الإنترنت واستخراج البيانات

**الميزات:**
- البحث في DuckDuckGo
- البحث في مواقع الصيدليات العربية
- استخراج البيانات من النتائج

**الدوال الرئيسية:**
- `search_web()` - البحث العام
- `search_pharmacy_sites()` - البحث في مواقع الصيدليات
- `search_and_scrape()` - البحث واستخراج البيانات

---

#### 6. `web_scraper.py` ⭐⭐⭐ **استخراج البيانات من المواقع**
**الوظيفة:** استخراج البيانات من صفحات الويب

**الميزات:**
- دعم مواقع متعددة (ويب طب، ويكيبيديا، صيدليات)
- استخراج العنوان، الوصف، المحتوى، السعر
- معالجة خاصة لكل موقع

**الدوال الرئيسية:**
- `scrape_url()` - استخراج بيانات من رابط
- `_extract_title()` - استخراج العنوان
- `_extract_content()` - استخراج المحتوى
- `_extract_price()` - استخراج السعر

---

#### 7. `pharmacy_sites_config.py` ⭐⭐⭐ **إعدادات مواقع الصيدليات**
**الوظيفة:** قائمة مواقع الصيدليات العربية

**المواقع المدعومة:**
- صحتك (sehetak.com) - مصر
- دواي (dawaey.com) - مصر
- صيدلية النهدي (nahdionline.com) - السعودية
- ويب طب (webteb.com) - عربي
- الطبي (altibbi.com) - عربي
- وغيرها (10+ موقع)

**الدوال:**
- `get_pharmacy_site_config()` - الحصول على إعدادات موقع
- `is_pharmacy_site()` - التحقق من موقع صيدلية
- `get_search_urls()` - الحصول على روابط البحث

---

#### 8. `nlp_preprocessing.py` ⭐⭐ **معالجة النصوص**
**الوظيفة:** معالجة وتحليل النصوص

**الميزات:**
- تنظيف النصوص
- Tokenization
- إزالة Stopwords
- تحليل المشاعر
- اكتشاف اللغة

**ملاحظة:** يعمل بدون nltk (fallback mode)

---

#### 9. `training_enhancer.py` ⭐⭐ **تحسين البيانات التدريبية**
**الوظيفة:** تحسين جودة البيانات التدريبية

**الميزات:**
- حساب درجة الجودة
- فلترة العينات عالية الجودة
- تنظيف البيانات

---

#### 10. `response_quality_improver.py` ⭐⭐ **تحسين جودة الردود**
**الوظيفة:** تحليل وتحسين الردود تلقائياً

**الميزات:**
- تحليل جودة الرد
- تحسين الرد تلقائياً
- حفظ العينات عالية الجودة فقط

---

#### 11. `rag_enhancer.py` ⭐⭐ **تحسين البحث والاسترجاع**
**الوظيفة:** تحسين البحث من مصادر متعددة

**الميزات:**
- توسيع الاستعلامات
- بحث من مصادر متعددة
- ترتيب النتائج حسب الجودة

---

#### 12. `training_data_exporter.py` ⭐ **تصدير البيانات**
**الوظيفة:** تصدير البيانات التدريبية
**الصيغ:** OpenAI Fine-tuning, ChatGPT Training

---

#### 13. `conversation_memory.py` ⭐ **ذاكرة المحادثة**
**الوظيفة:** حفظ ذاكرة المحادثات
**الميزات:** حفظ تاريخ المحادثة للسياق

---
