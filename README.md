# 🤖 Eva Chatbot - نظام شات بوت ذكي متكامل

## 📋 نظرة عامة

نظام شات بوت متقدم يدعم اللغة العربية والإنجليزية، مع ميزات البحث في الإنترنت، جمع البيانات من مواقع الصيدليات، وتحسين تلقائي للردود.

---

## 🚀 التشغيل السريع

### شغّل كل شيء بضغطة واحدة:
```bash
RUN.bat
```

**هذا كل شيء!** 🎉

بعد التشغيل، افتح: **http://localhost:8000**

---

## 📁 شرح الملفات

### 🔧 ملفات التشغيل

#### `RUN.bat` ⭐ **الملف الرئيسي للتشغيل**
- **الوظيفة:** تشغيل التطبيق الكامل (FastAPI + React)
- **الاستخدام:** اضغط نقرتين عليه
- **ما يفعله:**
  - يتحقق من Python والبيئة الافتراضية
  - يثبت المتطلبات تلقائياً
  - يشغل FastAPI على المنفذ 8000
  - يخدم React build (إن وجد)

#### `STOP.bat`
- **الوظيفة:** إيقاف جميع السيرفرات
- **الاستخدام:** اضغط نقرتين لإيقاف كل شيء

---

### 🐍 ملفات Python الأساسية

#### `fastapi_app.py` ⭐ **الملف الرئيسي للسيرفر**
- **الوظيفة:** FastAPI server مع جميع API endpoints
- **الميزات:**
  - `/api/chat` - محادثة مع الشات بوت
  - `/api/search` - البحث في الإنترنت
  - `/api/training/*` - إدارة بيانات التدريب
  - `/api/health` - فحص الحالة
- **الاعتماديات:** `core_integration`, `web_search_service`, `database`

#### `core_integration.py` ⭐ **نظام التكامل المركزي**
- **الوظيفة:** ربط جميع المكونات ببعضها
- **المكونات:**
  - Database
  - Chatbot
  - Web Scraper
  - Web Search Service
  - Knowledge Base
  - Conversation Memory
- **الاستخدام:** يتم استدعاؤه تلقائياً من `fastapi_app.py`

#### `advanced_chatbot_integration.py` ⭐ **الشات بوت الرئيسي**
- **الوظيفة:** معالجة الرسائل وتوليد الردود
- **الميزات:**
  - معالجة اللغة العربية والإنجليزية
  - RAG (Retrieval Augmented Generation)
  - البحث في قاعدة البيانات والإنترنت
  - تحسين الردود تلقائياً
- **الاعتماديات:** `nlp_preprocessing`, `llm_integration`, `conversation_memory`

#### `database.py` ⭐ **قاعدة البيانات**
- **الوظيفة:** إدارة قاعدة البيانات SQLite
- **الجداول:**
  - `conversations` - المحادثات
  - `training_samples` - عينات التدريب
  - `medical_data` - البيانات الطبية
  - `scraped_data` - البيانات المستخرجة
  - `users` - المستخدمين
- **الاعتماديات:** `sqlalchemy`

---

### 🔍 ملفات البحث والاستخراج

#### `web_search_service.py` ⭐ **خدمة البحث في الإنترنت**
- **الوظيفة:** البحث في الإنترنت واستخراج البيانات
- **الميزات:**
  - البحث في DuckDuckGo
  - البحث في مواقع الصيدليات
  - استخراج البيانات من النتائج
- **الاعتماديات:** `ddgs`, `web_scraper`

#### `web_scraper.py` ⭐ **استخراج البيانات من المواقع**
- **الوظيفة:** استخراج البيانات من صفحات الويب
- **الميزات:**
  - دعم مواقع متعددة (ويب طب، ويكيبيديا، صيدليات)
  - استخراج العنوان، الوصف، المحتوى، السعر
  - معالجة خاصة لكل موقع
- **الاعتماديات:** `beautifulsoup4`, `requests`

#### `pharmacy_sites_config.py` ⭐ **إعدادات مواقع الصيدليات**
- **الوظيفة:** قائمة مواقع الصيدليات العربية
- **المواقع المدعومة:**
  - صحتك (sehetak.com)
  - دواي (dawaey.com)
  - صيدلية النهدي (nahdionline.com)
  - ويب طب (webteb.com)
  - الطبي (altibbi.com)
  - وغيرها
- **الاستخدام:** يتم استدعاؤه تلقائياً من `web_search_service.py`

---

### 🧠 ملفات الذكاء الاصطناعي

#### `nlp_preprocessing.py` ⭐ **معالجة النصوص**
- **الوظيفة:** معالجة وتحليل النصوص
- **الميزات:**
  - تنظيف النصوص
  - Tokenization
  - إزالة Stopwords
  - Lemmatization
  - تحليل المشاعر
  - اكتشاف اللغة
- **الاعتماديات:** `nltk` (اختياري), `spacy` (اختياري)

#### `llm_integration.py`
- **الوظيفة:** التكامل مع نماذج LLM
- **الميزات:** دعم OpenAI, HuggingFace, Ollama

#### `ai_knowledge_base.py`
- **الوظيفة:** قاعدة معرفة للشات بوت
- **المحتوى:** معلومات طبية وصيدلانية

#### `conversation_memory.py`
- **الوظيفة:** حفظ ذاكرة المحادثات
- **الميزات:** حفظ تاريخ المحادثة للسياق

---

### 📊 ملفات التدريب والتحسين

#### `training_enhancer.py` ⭐ **تحسين البيانات التدريبية**
- **الوظيفة:** تحسين جودة البيانات التدريبية
- **الميزات:**
  - حساب درجة الجودة
  - فلترة العينات عالية الجودة
  - تنظيف البيانات
- **الاستخدام:** يتم استدعاؤه تلقائياً

#### `response_quality_improver.py` ⭐ **تحسين جودة الردود**
- **الوظيفة:** تحليل وتحسين الردود تلقائياً
- **الميزات:**
  - تحليل جودة الرد
  - تحسين الرد تلقائياً
  - حفظ العينات عالية الجودة فقط
- **الاستخدام:** يتم استدعاؤه تلقائياً من `fastapi_app.py`

#### `rag_enhancer.py` ⭐ **تحسين البحث والاسترجاع**
- **الوظيفة:** تحسين البحث من مصادر متعددة
- **الميزات:**
  - توسيع الاستعلامات
  - بحث من مصادر متعددة
  - ترتيب النتائج حسب الجودة
- **الاستخدام:** يتم استدعاؤه تلقائياً من `advanced_chatbot_integration.py`

#### `training_data_exporter.py`
- **الوظيفة:** تصدير البيانات التدريبية
- **الصيغ المدعومة:** OpenAI Fine-tuning, ChatGPT Training

## 📦 المتطلبات

### Python:
- Python 3.8+
- Virtual environment (myenv)

### الحزم الأساسية:
- `fastapi` - السيرفر
- `uvicorn` - ASGI server
- `sqlalchemy` - قاعدة البيانات
- `beautifulsoup4` - استخراج البيانات
- `requests` - طلبات HTTP
- `ddgs` - البحث في DuckDuckGo

### الحزم الاختيارية:
- `nltk` - معالجة النصوص المتقدمة
- `spacy` - NLP متقدم
